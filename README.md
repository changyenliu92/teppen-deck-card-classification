# teppen-deck-predict

This project aims to build a machine learning model that can predict cards from the mobile game, Teppen.

Stretch goals:
1. Extracting cards from a deck screenshot that consists 30 cards.
2. High accuracy rate from predicting all the cards from Teppen coreset.
3. Model generates json object as an output for other services to read.
